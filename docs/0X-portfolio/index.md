# Final Project

This project should demonstrate your understanding of deep learning through a practical computer vision application.  
Your work should be clear, focused, and strong enough to include in your portfolio.

## Key Objectives

- Apply deep learning to a real-world problem.
- Use CNNs effectively for image-based tasks.
- Use object detection if it fits the problem.
- Present results clearly and professionally.
- Build something small, complete, and meaningful.

## Minimum Requirements

Your project should include:

- A clear problem statement
- A dataset
- Few CNN-based model
- Training and evaluation
- Metrics and example predictions
- A short discussion of results, limitations, and possible improvements

## Portfolio Project Ideas

Choose one idea below or propose a similar one.  
Projects are grouped by whether they are mainly **CNN-based** or mainly **object detection**.

### Mostly CNN-Based Projects

#### 1. Plant Disease Diagnosis
Build a model that classifies crop leaf images by disease type or severity level.  
This has clear agricultural value because it can support early diagnosis and help farmers respond faster.  
A strong version of this project could compare a simple CNN with a transfer learning model and analyze difficult cases.

#### 2. Medical Image Classification
Develop a model to classify medical images such as chest X-rays, skin lesion images, or retinal scans.  
This is a strong portfolio project because it connects deep learning to healthcare and requires careful evaluation.  
A good project should focus not only on accuracy, but also on class imbalance, false negatives, and visual explanation of results.

#### 3. Waste Material Classification
Create a classifier that recognizes categories such as plastic, paper, glass, metal, and organic waste.  
This is a realistic sustainability project with practical relevance for recycling systems and smart bins.  
You can strengthen it by testing data augmentation, transfer learning, and confusion between visually similar classes.

#### 4. Fruit Quality or Ripeness Classification
Train a model to classify fruit by freshness, ripeness, or visible defects.  
This is useful for food quality control, agriculture, and retail automation.  
A good extension is to compare performance under clean images versus more realistic lighting and background conditions.

#### 5. Animal Species Classification
Build a classifier for wildlife or domestic animal species from camera trap or field images.  
This is interesting because it connects computer vision with biodiversity monitoring and ecology.  
A strong project could study how performance changes across species with few samples or visually similar classes.

#### 6. Traffic Sign Recognition
Develop a model that classifies traffic signs from road images.  
This is a classic but still meaningful application because it connects directly to driver assistance and autonomous systems.  
To make it stronger, focus on robustness to blur, distance, weather, and poor lighting.

### Mostly Object Detection Projects

#### 1. Vehicle Detection and Counting
Build a system that detects vehicles in road or parking camera footage and estimates counts over time.  
This is a realistic smart city project with applications in traffic monitoring and planning.  
A strong version could include class-specific counts, simple speed estimation, or analysis under crowded scenes.

#### 2. Product Detection on Shelves
Train a detector to locate products on supermarket or pharmacy shelves.  
This has real retail value for inventory monitoring, restocking, and shelf compliance.  
You can strengthen the project by detecting missing items, comparing detector performance, or handling occlusion and dense layouts.

#### 3. Parking Space Occupancy Detection
Create a model that detects whether parking spots are occupied or free from camera images.  
This is a practical problem in smart parking systems and works well as a focused portfolio project.  
A stronger version could combine object detection with a parking-slot logic layer to estimate occupancy rates.

#### 4. Helmet Detection for Safety Monitoring
Develop a detector that identifies whether motorcyclists or workers are wearing helmets.  
This is a highly relevant safety application for construction sites, factories, or road monitoring.  
A strong project could include challenging scenarios such as partial visibility, multiple people, and different camera angles.

#### 5. Mask or PPE Detection
Build a detection system for personal protective equipment such as masks, safety vests, or helmets.  
This project is useful in workplace safety and public health monitoring.  
It becomes more interesting when you evaluate performance across crowded scenes and different object sizes.

#### 6. Fruit or Crop Detection in Fields
Train a detector to locate fruits on trees or crops in field images.  
This connects computer vision to precision agriculture and can support yield estimation or automated harvesting.  
A strong version could measure count accuracy, missed detections, and performance in cluttered outdoor scenes.

>> Choose a project that is realistic, technically solid, and small enough to complete well. 
>> The goal is not only to train a model, but to build a project you would be comfortable showing in a portfolio.

## Deliverables

Submit:

- Code
- Short report

## What Makes a Strong Project

A strong project:

- Solves a clear and realistic problem
- Uses an appropriate dataset
- Shows a clean training and evaluation process
- Includes visual examples of results
- Explains what worked and what did not
- Looks professional enough to share publicly

## Notes

Keep the scope manageable.  
A smaller project done well is better than a large project that is incomplete.
